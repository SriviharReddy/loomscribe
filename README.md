# Vibe API Chatbot

A lightweight, vanilla HTML/CSS/JS frontend for a chatbot powered by the DeepSeek API. No heavy frameworks—just pure vibes.

## Features
- **Vanilla Stack**: Pure HTML, CSS, and JavaScript.
- **DeepSeek Integration**: Connects directly to the DeepSeek API.
- **Modern UI/UX**: Sleek, responsive design with smooth micro-animations.
